# 🌤 Weather App

This is a simple Weather App that fetches real-time weather data using an API.

---

## 🌐 Live Demo
🔗 https://rishu-raj-dev01.github.io/portfolio/

---

## 👨‍💻 About Project
This project allows users to enter a city name and get current weather details like temperature and weather condition.

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- OpenWeather API

---

## ⚙️ Features
- Search weather by city
- Real-time temperature display
- Weather condition display

---

## 📂 How It Works
- User enters city name
- App fetches data from OpenWeather API
- Displays weather information
